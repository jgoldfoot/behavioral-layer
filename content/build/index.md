The mechanisms that implement the behavioral layer. This section covers agent frameworks,
orchestration, memory, tool use, and harnesses, read for how they support (or undermine)
behavioral contracts, escalation, and repair.

## What belongs here

- Agent and multi-agent frameworks and orchestration patterns.
- Memory systems and state management for long-running agents.
- Tool use, function calling, and the protocols that standardize them.
- Harnesses and runtimes: the scaffolding around a model that turns it into an agent.

## What goes elsewhere

- Whether a mechanism produces trustworthy behavior is judged in
  [[evaluate/index|Evaluate]].
- The behavioral goals these mechanisms serve are defined in [[behavior/index|Behavior]].
